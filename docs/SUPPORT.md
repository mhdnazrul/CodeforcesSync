# Support

## Getting Help

If you need help with CodeforcesSync, here are the best ways to get it:

### Documentation

Start with the documentation in the `docs/` directory:

- [Troubleshooting](docs/Troubleshooting.md) — Common issues and solutions.
- [FAQ](docs/FAQ.md) — Frequently asked questions.
- [Deployment](docs/Deployment.md) — Setting up the OAuth broker.

### GitHub Issues

- **Bug reports** — Open a [bug report](https://github.com/mhdnazrul/CodeforcesSync/issues/new?template=bug_report.md).
- **Feature requests** — Open a [feature request](https://github.com/mhdnazrul/CodeforcesSync/issues/new?template=feature_request.md).
- **Questions** — Open a [question](https://github.com/mhdnazrul/CodeforcesSync/issues/new?template=question.md).

### Discussions

For general questions, ideas, or community support, use [GitHub Discussions](https://github.com/mhdnazrul/CodeforcesSync/discussions).

### Email

For security issues only: **mhdnazrul511@gmail.com** (see [SECURITY.md](SECURITY.md)).

## Before Opening an Issue

1. Check the [Troubleshooting](docs/Troubleshooting.md) guide.
2. Search [existing issues](https://github.com/mhdnazrul/CodeforcesSync/issues) to see if your problem has already been reported.
3. Make sure you are on the latest release version.

## Response Times

| Channel | Expected Response Time |
|---------|----------------------|
| GitHub Issues | 1-3 business days |
| GitHub Discussions | 1-5 business days |
| Security Email | Within 48 hours |
